Bootcamp 2020: Mirror Paint

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project demonstrates a comprehensive understanding of the use of loops, transformations, and user interaction. I created a canvas which mirrors the stroke of your brush radially, like a kaleidoscope. This was acheived by rotating the stroke in real time, while the mouse is being clicked. I also used Perlin noise on the hue for a rainbow stroke effect. You can change the number of 'sides', or times the brush gets mirrored, using the 'd' and 'f' keys, and the 's' and 'n' keys for saving and wiping the canvas.

Check out the code on Github: https://github.com/YFarah25/adbt212
